## Installation
Add this line to your application's Gemfile:

```ruby
gem 'sword'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install sword
```

Install configs
```bash
$ rails generate sword:config
```

Add routes to config/routes.rb
```ruby
mount Sword::Engine => "/sword"
```
